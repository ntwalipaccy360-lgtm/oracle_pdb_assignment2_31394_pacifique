
# Oracle PDB Assignment

## 1. Assignment Overview
This project demonstrates Oracle Multitenant architecture by creating and managing Pluggable Databases (PDBs), users, and Oracle Enterprise Manager (OEM).

## 2. Oracle Environment
- Oracle Database: 21c XE
- OS: Windows
- Tool: SQL*Plus, Oracle EM Express

## 3. Tasks Completed

### Task 1: PDB Creation and User Setup
- Created PDB
- Created user inside PDB
- Granted privileges
- Verified login

### Task 2: Temporary PDB Management
- Created temporary PDB
- Verified existence
- Opened and deleted PDB

### Task 3: Oracle Enterprise Manager
- Accessed EM Express
- Viewed database and PDB details

## 4. Challenges and Solutions
Faced issues with PDB not opening and resolved by correcting container and privilege settings.

## 5. Lessons Learned
Learned Oracle Multitenant architecture, PDB management, and OEM usage.
